# calculate_histogram

## 函数签名

```rust
pub fn calculate_histogram(gray: &GrayImage) -> [u32; 256]
```

## 功能描述

计算灰度图像的直方图，统计每个灰度级别（0-255）出现的像素数量。直方图是图像灰度分布的统计表示，广泛用于图像分析、对比度增强和阈值分割。

## 参数说明

- `gray: &GrayImage` - 输入的灰度图像引用

## 返回值

- `[u32; 256]` - 长度为 256 的数组，其中 `hist[i]` 表示灰度值为 `i` 的像素数量

## 数学原理

直方图函数定义为：

```
h(i) = Σ δ(f(x,y) - i)
```

其中：
- `h(i)` 是灰度级 `i` 的频次
- `f(x,y)` 是坐标 `(x,y)` 处的灰度值
- `δ()` 是 Kronecker delta 函数（相等时为1，否则为0）
- 求和遍历图像所有像素

## 实现逻辑

### 第 7 行：初始化直方图数组
```rust
let mut hist = [0u32; 256];
```
**原理**：创建长度为 256 的计数器数组，初始值全为 0。每个索引对应一个灰度级（0-255）。

---

### 第 8-10 行：遍历所有像素并统计
```rust
for pixel in gray.pixels() {
    hist[pixel[0] as usize] += 1;
}
```
**原理**：
- **第 8 行**：使用迭代器遍历图像的所有像素
- **第 9 行**：
  - `pixel[0]` 获取灰度值（`GrayImage` 只有一个通道）
  - `as usize` 将 `u8` 类型转换为数组索引
  - `hist[...]` 对应灰度级的计数器加 1
  
**时间复杂度**：O(W × H)，其中 W 和 H 分别是图像的宽度和高度

---

### 第 11 行：返回直方图
```rust
hist
```
返回统计完成的直方图数组。

## 应用场景

1. **对比度分析**：通过直方图分布判断图像对比度
   - 集中分布 → 低对比度
   - 均匀分布 → 高对比度

2. **阈值分割**：找到分割前景和背景的最佳阈值（如 Otsu 算法）

3. **直方图均衡化**：基于累积分布函数重新映射灰度值

4. **图像比较**：通过直方图相似度比较图像

## 使用示例

```rust
let gray_img = to_grayscale(&rgb_img);
let hist = calculate_histogram(&gray_img);

// 打印前 10 个灰度级的频次
for i in 0..10 {
    println!("灰度 {}: {} 个像素", i, hist[i]);
}

// 计算图像总像素数（验证）
let total: u32 = hist.iter().sum();
assert_eq!(total, gray_img.width() * gray_img.height());
```

## 性能说明

- 空间复杂度：O(1) - 固定大小数组（256 个元素）
- 时间复杂度：O(N) - N 为总像素数
- 无内存分配：使用栈上的固定数组
