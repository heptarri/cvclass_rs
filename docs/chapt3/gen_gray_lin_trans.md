# gen_gray_lin_trans

## 函数签名

```rust
pub fn gen_gray_lin_trans(src: &RgbImage, alpha: f32, beta: f32) -> GrayImage
```

## 功能描述

对图像进行线性灰度变换，根据公式 `y = α × x + β` 重新映射像素灰度值。用于调整图像的对比度和亮度。

## 参数说明

- `src: &RgbImage` - 输入的 RGB 彩色图像引用
- `alpha: f32` - 斜率参数（对比度因子）
  - `α > 1`：增强对比度
  - `α = 1`：保持对比度
  - `0 < α < 1`：降低对比度
- `beta: f32` - 截距参数（亮度偏移）
  - `β > 0`：增加亮度
  - `β = 0`：保持亮度
  - `β < 0`：降低亮度

## 返回值

- `GrayImage` - 变换后的灰度图像

## 数学原理

线性变换的通用形式：

```
g(x, y) = α · f(x, y) + β
```

其中：
- `f(x, y)` 是输入像素的灰度值（范围 [0, 255]）
- `g(x, y)` 是输出像素的灰度值
- `α` 控制动态范围的扩张或收缩
- `β` 控制整体亮度的偏移

最终需要将结果限制在 [0, 255] 范围内：

```
g(x, y) = clamp(α · f(x, y) + β, 0, 255)
```

### 对比度与亮度的关系

- **对比度**：由 `α` 控制
  - `α > 1`：拉伸灰度范围，暗的更暗，亮的更亮
  - `α < 1`：压缩灰度范围，减少明暗差异
  
- **亮度**：由 `β` 控制
  - 所有像素值同时增加或减少固定量

## 实现逻辑

### 第 43 行：转换为灰度图
```rust
let gray = to_grayscale(src);
```
**原理**：
- 使用加权平均法将 RGB 图像转换为灰度图
- 公式：`Gray = 0.299R + 0.587G + 0.114B`

---

### 第 44 行：创建目标图像
```rust
let mut dst = GrayImage::new(gray.width(), gray.height());
```
**原理**：
- 创建与输入相同尺寸的空白灰度图像
- 用于存储变换后的结果

---

### 第 45-51 行：遍历并应用线性变换
```rust
for y in 0..gray.height() {
    for x in 0..gray.width() {
        let value = gray.get_pixel(x, y)[0] as f32;
        let mapped = (alpha * value + beta).round();
        dst.put_pixel(x, y, Luma([mapped.clamp(0.0, 255.0) as u8]));
    }
}
```
**原理**：

- **第 45-46 行**：双重循环遍历所有像素坐标

- **第 47 行**：读取当前像素的灰度值
  - `[0]` 提取单通道灰度值
  - `as f32` 转换为浮点数以支持小数运算

- **第 48 行**：应用线性变换公式
  - 计算：`mapped = α × value + β`
  - `.round()` 四舍五入到最近的整数
  - 示例：
    - 输入 `value = 100`, `α = 1.5`, `β = 20`
    - 计算：`1.5 × 100 + 20 = 170`

- **第 49 行**：限制范围并写入结果
  - `.clamp(0.0, 255.0)` 将值限制在合法范围
    - 小于 0 的值设为 0
    - 大于 255 的值设为 255
  - `as u8` 转换为无符号 8 位整数
  - `Luma([...])` 包装为灰度像素
  - `put_pixel()` 写入目标图像

**边界处理示例**：
- 输入 100，`α=2.0`，`β=100` → `100×2+100=300` → clamp → 255
- 输入 100，`α=0.5`，`β=-100` → `100×0.5-100=-50` → clamp → 0

---

### 第 52 行：返回结果
```rust
dst
```
返回变换后的灰度图像。

## 使用示例

### 示例 1：增强对比度
```rust
// α=1.5 增强对比度，β=0 保持亮度
let enhanced = gen_gray_lin_trans(&img, 1.5, 0.0);
```

### 示例 2：增加亮度
```rust
// α=1.0 保持对比度，β=50 增加亮度
let brighter = gen_gray_lin_trans(&img, 1.0, 50.0);
```

### 示例 3：同时调整
```rust
// α=1.2 略微增强对比度，β=20 略微增加亮度
let adjusted = gen_gray_lin_trans(&img, 1.2, 20.0);
```

### 示例 4：反转图像
```rust
// α=-1, β=255 实现灰度反转
let inverted = gen_gray_lin_trans(&img, -1.0, 255.0);
// 公式：y = -x + 255，将 0 变为 255，255 变为 0
```

## 应用场景

1. **曝光校正**：修正过曝或欠曝的图像
2. **对比度拉伸**：扩展窄范围的灰度分布
3. **图像标准化**：将不同光照条件的图像统一到相同范围
4. **预处理**：为后续算法（如边缘检测）提供更好的输入

## 性能说明

- 时间复杂度：O(W × H)
- 空间复杂度：O(W × H) - 需要存储输出图像
- 每像素操作：1 次乘法、1 次加法、1 次 clamp
